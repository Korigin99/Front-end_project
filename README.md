   ```

    최대한 활용하여 사이트를 제작하는 것이었기 때문에, 가장 기술을 많이 
    활용할 수 있는 게임을 만들고자 주제를 정했습니다.
    ```
- 프로젝트 주제
    ```
    html, css, js를 활용한 게임을 만들고 이를 모아놓은 스낵게임 사이트를 
    만들어 구동 시키기
    ```
